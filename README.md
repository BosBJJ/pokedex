
# Pokedex


## About

This project uses the PokeAPI to let you explore Pokemon world data from your terminal. You can browse map areas, explore locations, catch Pokemon, inspect the ones you’ve caught, and list them in your Pokedex.

## Features

- Browse location areas
- Page forward and backward through map results
- Explore an area to find wild Pokemon
- Attempt to catch Pokemon
- Inspect caught Pokemon
- List all caught Pokemon in your Pokedex

## Commands

| Command | Description |
|---|---|
| `help` | Show available commands |
| `map` | Show the next page of location areas |
| `mapb` | Show the previous page of location areas |
| `explore <area>` | Show Pokemon in a location area |
| `catch <pokemon>` | Attempt to catch a Pokemon |
| `inspect <pokemon>` | Show details for a caught Pokemon |
| `pokedex` | List all caught Pokemon |
| `exit` | Exit the program |

## Run locally

```bash
go run .
```
